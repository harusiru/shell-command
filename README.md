### -Various Command Commands-
- cd: change directory
- ls: list files and directories 
ex:
```sh
$ pwd
/home/user name
$ ls
~ ~ ~ ~ abcd ~ ~
$ cd abcd
$ pwd
/home/user name/abcd
$ ls
0123
$ cd..
$ pwd
/home/user name
$ ls -l
(total file)
```
